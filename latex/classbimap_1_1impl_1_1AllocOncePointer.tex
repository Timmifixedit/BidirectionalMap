\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class bimap\+::impl\+::\+Alloc\+Once\+Pointer$<$ T $>$}

Very simple pointer class that can be used to allocate storage once but can also be used as a non owning pointer. 

Unlike shared\+\_\+ptr, copies of this class are non-\/owning pointers and unlike weak\+\_\+ptr, non-\/owning pointers do not know if the object behind the pointer still exists. The owning pointer deallocates storage at destruction

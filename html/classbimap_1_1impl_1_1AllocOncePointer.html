Detailed Description

template<typename T>
class bimap::impl::AllocOncePointer< T >

Very simple pointer class that can be used to allocate storage once but can also be used as a non owning pointer. Unlike shared_ptr, copies of this class are non-owning pointers and unlike weak_ptr, non-owning pointers do not know if the object behind the pointer still exists. The owning pointer deallocates storage at destruction
